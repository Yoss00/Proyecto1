SQL, Structure Query Language (Lenguaje de Consulta Estructurado) es un lenguaje de programacion para trabajar con Base de Datos Relacionales como MySQL, Oracle, etc.
MySQL es un interpretador de SQL, es un servidor de Base de Datos.
MySQL permite crear Base de Datos y tablas, insertar datos, modificarlos, eliminarlos, ordenarlos, hacer consultas y realizar muchas operaciones, etc., resumiendo; administrar Base de Datos

Este tutorial tiene por objetivo acercar los conceptos iniciales para introducirse en el mundo de las Base de Datos.
